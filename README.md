## ✨ 2026-05-08 识别精度与性能优化

### 🔬 两阶段 OCR 识别方案
- **专用 OCR 先行**：集成 Google ML Kit 中文 OCR 引擎，精准提取药盒/标签上的原始文字。
- **大模型结构化**：Gemma 4 多模态以 OCR 文字为主要依据、图片为辅助验证，输出结构化用药 JSON。
- **双模式自动回退**：多模态就绪时传图+OCR 文字；iOS 等图片受限场景自动切纯文本模式，仅靠 OCR 文字完成结构化。
- **过程透明可校对**：识别面板实时展示 OCR 提取原文，用户可肉眼校对后再确认保存。

### ⚡ iOS 性能与发热优化
- **推理参数精简化**：识药场景 maxTokens 256（原 2048）、temperature 0.1（原 0.7），单次推理负载降至原来的 1/8。
- **用后即释放**：识药完成后立即关闭推理模型，GPU 不常驻占用，降低持续发热。
- **flutter_gemma 0.14.5**：iOS 原生库体积减少 63%，含 App Store 上架必需修复及 dylib 加载稳定性改进。
- **Metal GPU 加速**：iOS 真机完整支持 Metal 后端，模拟器自动退至 CPU。
