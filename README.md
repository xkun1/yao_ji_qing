# 💊 药记清 (Yao Ji Qing)

**准时服药，不负爱与嘱托。**

每一份医嘱都承载着康复的希望，每一声提醒都传递着家人的牵挂。药记清，让健康准时，让爱不缺席。

## ✨ 2026-04-16 重大更新日志 (Milestone)

### 🤖 顶配端侧 AI 革命
- **Gemma 4 (E2B) 全面适配**：集成 Google 最新一代端侧大模型，实现 100% 离线隐私保护。
- **多模态视觉识别**：废弃传统 OCR，直接将原图喂给 AI，识别率提升一个量级。
- **打字机式实时反馈**：识别过程中实时展示 AI 思维流，并支持长文本自动滚动。
- **智能硬件适配**：根据手机芯片（高通/麒麟/联发科）自动切换 CPU/NPU/GPU 运算后端。

### ⏰ 系统级闹铃与保活
- **核武器级保活**：引入 `setAlarmClock` 与 `onTaskRemoved` 监听，实现杀进程、划通知后的“秒级复活”。
- **系统闹钟联动**：采用系统默认闹钟铃声，并支持锁屏状态下的全屏唤醒提醒。
- **智能守护通知**：状态栏实时显示“下一顿该吃什么”，今日任务完成后自动消失，拒绝占位。

### 🎨 交互与美学升级
- **极致丝滑动画**：首页加号菜单实现“从左向右渐次弹出”的弹性动效，开合逻辑独立优化。
- **健康名言库**：首页头部随机展示精选健康语录，支持自适应折行。
- **专业设置体系**：新增“AI 引擎管理（下载/清理）”、“权限设置引导”、“关于”及“合规政策”页面。
- **专注列表**：已服用的药品自动从首页消失，呈现纯净的“待办清单”。

## 🚀 核心特性

- **📸 拍照智能识别**：Gemma 4 多模态驱动，医嘱一拍即得。
- **💬 咨询药师**：集成离线 AI 聊天功能，随时解答用药禁忌。
- **📊 健康统计**：按时间段（早中晚夜）分类统计服药遵从率。
- **🛡️ 隐私至上**：除 AI 识别过程外，数据 100% 本地存储，无云端同步。

## 🛠 关键权限设置 (必读)

为了确保 100% 准时提醒，请按照 **“设置 -> 保活与权限管理”** 中的引导，开启以下权限：
1. **允许后台活动**：开启自启动、关联启动。
2. **忽略电池优化**：允许 App 后台常驻。
3. **通知与闹钟**：允许开启精确闹钟与全屏提醒。

## 📦 技术栈

- **Core**: Flutter (Dart)
- **Local AI**: Flutter Gemma (Gemma 4 E2B)
- **Database**: Isar NoSQL
- **Storage**: path_provider + shared_preferences
- **Network**: background_downloader (后台断点续传)
